## Commands

* Generate MCP server JSON file: `fastmcp install mcp-json app.py --project .`
* Inspect MCP server: `fastmcp inspect --format mcp -o manifest.json`
* Call tool: `fastmcp call server.py greet name=World`
* List resources and prompts: `fastmcp list app.py --resources --prompts`
* Inspector: `npx @modelcontextprotocol/inspector uv --directory /path/to/pdfapi/ run fastmcp run app.py`
